# Managing Static Pages in Strapi CMS

Strapi CMS allows you to easily manage and create static pages for your website. These pages can include important information like terms and conditions, privacy policies, about us, affiliate terms, and more. This guide will walk you through how to control and manage these static pages using Strapi CMS.

## Creating a New Static Page

To create a new static page in Strapi CMS, follow these simple steps:

<Steps>
  <Step title="Access the Content Manager">
    After logging in, click on "Content Manager" in the dashboard's sidebar. This is where you can manage all your content types.
  </Step>
  <Step title="Click on 'Static Pages'">
    In the Content Manager, locate and click on "StaticPage" in the list of content types.
  </Step>
  <Step title="Create a New Page Entry">
    To create a new static page, click the "Add New Static Page" button.
  </Step>
  <Step title="Fill in the Details">
    Fill in the following details for your new page:
    - **Title**: Enter the title for your new static page. This title will be displayed on your website.
    - **URL**: Set the URL for your new page. It determines the page's web address.
    - **Static Content**: Here, you can add the content for your static page using the WYSIWYG (What You See Is What You Get) editor. This editor allows you to format your text, add images, and create the content for your page. If you need more than 1 container just click on "+"
  </Step>
  <Step title="Save Your Page">
    After entering the details, click the "Save" button to create your new static page.
  </Step>
</Steps>

## Modifying an Existing Static Page

If you need to make changes to an existing static page, follow these steps:

<Steps>
  <Step title="Access the Content Manager">
    Log in to your Strapi CMS account and navigate to the Content Manager by clicking "Content Manager" in the dashboard sidebar.
  </Step>
  <Step title="Locate Your Static Page">
    In the Content Manager, find and click on "StaticPage" to see the list of existing pages.
  </Step>
  <Step title="Select the Page to Edit">
    Locate the static page you want to edit and click on it.
  </Step>
  <Step title="Edit Page Properties">
    You can edit the following properties of the page:
    - **Title**: You can update the page's title to reflect any changes.
    - **URL**: Modify the URL if needed.
    - **Static Content**: Edit the content of the page using the WYSIWYG editor.
  </Step>
  <Step title="Save Your Changes">
    After making your edits, be sure to save your changes by clicking the "Save" button.
  </Step>
</Steps>

## Deleting a Static Page
<Warning>
  Please be aware that deleting a "Static Page" is permanent and cannot be undone. Make sure you no longer need the item before proceeding with the deletion.
</Warning>

If you need to delete a static page, follow these steps:

<Steps>
  <Step title="Access the Content Manager">
    Log in to your Strapi CMS account and access the Content Manager by clicking "Content Manager" in the dashboard sidebar.
  </Step>
  <Step title="Find Your Static Page">
    In the Content Manager, locate and click on "Static Pages" to view the list of existing pages.
  </Step>
  <Step title="Select the Page to Delete">
    Find the static page you want to delete and click on it.
  </Step>
  <Step title="Delete the Page">
    In the page's detail view, look for the "Delete" or "Trash" button.
  </Step>
  <Step title="Confirm Deletion">
    When prompted, confirm the deletion. Please note that this action is permanent and cannot be undone. Ensure that you no longer need the page before confirming the deletion.
  </Step>
</Steps>

<Warning>
  Please be aware that any changes to this content type will require a website redeploy, which typically takes around 2 to 3 minutes.
</Warning>

With these instructions, you can effectively manage and control your static pages in Strapi CMS, making it easy to create, modify, and delete content as needed.
