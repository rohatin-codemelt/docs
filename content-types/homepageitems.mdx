# Managing Home Page Items in Strapi CMS

Home Page Items are used to manage various elements on your home page. You can choose different types of content for these items, each with its unique display style. This guide will help you understand how to use the "HomePageItem" content type effectively.

</Tip>

2. Title

The "Title" field allows you to set the title that will be displayed for the selected card type from the "Type" dropdown. This title helps users understand the content or feature being presented.

3. Icon (Optional)

The "Icon" field is optional and allows you to display an icon before the title. This can help enhance the visual representation of the content or feature.

4. Position

The "Position" field uses the drag and drop plugin to determine the order of items on the page. You can easily reorder the items by dragging and dropping them in the Content Manager.

By using the "HomePageItems" content type with these fields, you can effectively manage and arrange various content elements on your home page to provide an engaging and informative experience for your users.

## Creating a New Home Page Item

To add a new item to your home page, follow these steps:

<Steps>
  <Step title="Access Content Manager">
    In the Content Manager, locate and click on "HomePageItems" in the list of content types.
  </Step>
  <Step title="Create a New Home Page Item">
    You'll see a list of existing Home Page Items. To create a new one, click the "Add New Home Page Item" button.
  </Step>
  <Step title="Fill in Details">
    Fill in the following details for your new item:
    - **Type**: Select the type of content you want to add. Options include EGT Jackpot, Manhattan Jackpot, Popular Matches, Promotion Slider, Top Winners, Carousel, or Game Category. If you choose "Game Category," additional fields are required.
    - **Title**: Enter the title to be displayed for the selected card type from the "Type" dropdown.
    - **Icon** (Optional): Add an icon that will be displayed before the title.
    - **Position**: Specify the order in which this item should appear on your home page.
    - If you selected "Game Category," you need to provide:
      - **EM_GameGroup**: Specify the EveryMatrix Game Group for this custom slider.
      - **SeeMorePath**: Define the path to see more details about the games in this category.
  </Step>
  <Step title="Save Your Home Page Item">
    After filling in the information, click the "Save" button to create the new Home Page Item.
  </Step>
</Steps>

## Editing an Existing Home Page Item

If you need to make changes to an existing Home Page Item, follow these steps:

<Steps>
  <Step title="Access Content Manager">
    In the Content Manager, find and click on "HomePageItems" in the list of content types.
  </Step>
  <Step title="Select Item to Edit">
    You'll see a list of existing Home Page Items. Locate the item you want to edit and click on it.
  </Step>
  <Step title="Edit Item Properties">
    Once you've selected the item, you can edit the properties as needed:
    - **Type**: Update the type of content for this item by selecting a different option from the "Type" dropdown.
    - **Title**: Edit the title displayed for this item.
    - **Icon**: Update the icon or remove it if it's no longer needed.
    - **Position**: Modify the order in which this item appears on the home page.
    - If the item is of "Game Category" type, remember to update the "EM_GameGroup" and "SeeMorePath" fields.
  </Step>
  <Step title="Save Your Edits">
    After making your edits, remember to save your changes by clicking the "Save" button.
  </Step>
</Steps>

## Deleting a  Home Page Item
<Warning>
  Please be aware that deleting a Home Page Item" is permanent and cannot be undone. Make sure you no longer need the item before proceeding with the deletion.
</Warning>

If you need to remove a "Home Page Item" from your  live home page, follow these steps:

<Steps>
  <Step title="Access Content Manager">
    In the Content Manager, find and click on "HomePageItem" in the list of content types.
  </Step>
  <Step title="Select Item to Delete">
    You'll see a list of existing Burger Menu Items. Locate the item you want to delete and click on it.
  </Step>
  <Step title="Find the Delete Button">
    In the item's detail view, find the "Delete" or "Trash" button.
  </Step>
  <Step title="Confirm Deletion">
    Confirm the deletion when prompted. Please be aware that deleting a "Home Page Item" is permanent and cannot be undone. Make sure you no longer need the item before proceeding with the deletion.
  </Step>
</Steps>

<Tip>
  The updated content will be reflected in the burger menu right away. If it's not updating, you can perform a hard reload by pressing "Ctrl + Shift + R."
</Tip>
By following these steps, you can easily edit existing Home Page Items to keep your home page informative and engaging for your users.
