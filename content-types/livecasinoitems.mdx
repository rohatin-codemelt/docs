2. Title

The "Title" field allows you to set the title that will be displayed for the selected card type from the "Type" dropdown. This title helps users understand the content or feature being presented.

3. Icon (Optional)

The "Icon" field is optional and allows you to display an icon before the title. This can help enhance the visual representation of the content or feature.

4. Position

The "Position" field uses the drag and drop plugin to determine the order of items on the page. You can easily reorder the items by dragging and dropping them in the Content Manager.

By using the "CasinoItems" content type with these fields, you can effectively manage and arrange various content elements on your  live casino page to provide an engaging and informative experience for your users.

## Creating a New  Live Casino Item

To add a new item to your  live casino page, follow these steps:

1. In the Content Manager, locate and click on "CasinoItems" in the list of content types.

2. You'll see a list of existing  Live Casino Items. To create a new one, click the "Add New  Live Casino Item" button.

3. Fill in the following details for your new item:

   - **Type**: Select the type of content you want to add. You can choose from options like EGT Jackpot, Manhattan Jackpot, Popular Matches, Promotion Slider, Top Winners, Carousel, or Game Category. If you choose "Game Category," additional fields are required.

   - **Title**: Enter the title to be displayed for the selected card type from the "Type" dropdown.

   - **Icon** (Optional): You can add an icon that will be displayed before the title.

   - **Position**: Specify the order in which this item should appear on your  live casino page.

   - If you selected "Game Category," you need to provide:
     - **GameCategory**: Specify the EveryMatrix Game Group for this custom slider.
     - **SeeMorePath**: Define the path to see more details about the games in this category.

4. After filling in the information, click the "Save" button to create the new  Live Casino Item.

## Editing an Existing  Live Casino Item

If you need to make changes to an existing  Live Casino Item, follow these steps:

1. In the Content Manager, find and click on "CasinoItems" in the list of content types.

2. You'll see a list of existing  Live Casino Items. Locate the item you want to edit and click on it.

3. Once you've selected the item, you can edit the properties as needed:

   - **Type**: You can update the type of content for this item by selecting a different option from the "Type" dropdown.

   - **Title**: Edit the title displayed for this item.

   - **Icon**: You can update the icon or remove it if it's no longer needed.

   - **Position**: Modify the order in which this item appears on the  live casino page.

   - If the item is of "Game Category" type, remember to update the "GameCategory" and "SeeMorePath" fields.

4. After making your edits, remember to save your changes by clicking the "Save" button.

## Deleting a Live Casino Item

If you need to remove a "Live Casino Item" from your  live casino page, follow these steps:

1. In the Content Manager, locate and click on "LiveCasinoItems" in the list of content types.

2. You'll see a list of existing  Live Casino Items. Find the item you want to delete and click on it.

3. In the item's detail view, find the "Delete" or "Trash" button.

4. Confirm the deletion when prompted.

Please be aware that deleting a "Live Casino Item" is permanent and cannot be undone. Make sure you no longer need the item before proceeding with the deletion.

5. The updated content will be reflected on your  live casino page right away. If it's not updating, you can perform a hard reload by pressing "Ctrl + Shift + R."

By following these steps, you can easily edit existing  Live Casino Items to keep your  live casino page informative and engaging for your users.
