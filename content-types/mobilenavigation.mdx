## Create a New Menu Item 
<Steps>
  <Step title="Access the Content Manager">
    Access the Content Manager in Strapi by logging in to your Strapi CMS account.
  </Step>
  <Step title="Select 'FooterMobileMenuItem'">
    In the Content Manager, find and click on "FooterMobileMenuItem" in the list of content types.
  </Step>
  <Step title="Create a new Menu Item">
    To add a new navigation item to the mobile menu, click the "Add New Footer Mobile Menu Item" button.
  </Step>
  <Step title="Fill in the Details">
    Fill in the following details for your new Promotion:
    - **Title**: Enter the title for the menu item.
    - **Icon**: Choose or upload an icon that represents the menu item (e.g., an arrow or symbol).
    - **Path**: Specify the path or URL that the menu item should navigate to when clicked.
    - **Position**: Define the order in which this menu item should appear in the mobile menu.
  </Step>
  <Step title="Saving the changes">
    After entering all the information, click the "Save" button to create your new Menu Item.
  </Step>
</Steps>

## Modify a Menu Item 
<Steps>
  <Step title="Access the Content Manager">
    Access the Content Manager in Strapi by logging in to your Strapi CMS account.
  </Step>
  <Step title="Select 'FooterMobileMenuItem'">
    In the Content Manager, find and click on "FooterMobileMenuItem" in the list of content types.
  </Step>
  <Step title="Locate the Menu Item to Edit">
    You'll see a list of existing mobile menu items. Locate the menu item you want to modify and click on it.
  </Step>
  <Step title="Fill in the Details">
    Edit in the following details for your new Promotion:
    - **Title**: Enter the title for the menu item.
    - **Icon**: Choose or upload an icon that represents the menu item (e.g., an arrow or symbol).
    - **Path**: Specify the path or URL that the menu item should navigate to when clicked.
    - **Position**: Define the order in which this menu item should appear in the mobile menu.
  </Step>
  <Step title="Saving the changes">
    After entering all the information, click the "Save" button to create your new Menu Item.
  </Step>
</Steps>

## Delete a Menu Item

<Warning>
  Please be aware that deleting a "FooterMobileMenuItem" is permanent and cannot be undone. Make sure you no longer need the item before proceeding with the deletion.
</Warning>
If you no longer need a Footer Mobile Menu Item and wish to delete it, follow these steps:

<Steps>
  <Step title="Access FooterMobileMenuItem in the Content Manager">
    In the Content Manager, find and click on "FooterMobileMenuItem" in the list of content types.
  </Step>
  <Step title="Select the Item to Delete">
    You'll see a list of existing Footer Mobile Menu Item.
  </Step>
  <Step title="Delete the Item">
    In the category's detail view, look for the "Delete" or "Trash" button.
  </Step>
  <Step title="Confirm Deletion">
    When prompted, confirm the deletion. Keep in mind that deleting a Footer mobile Item is permanent and cannot be undone, so ensure that you no longer need the item before confirming the deletion.
  </Step>
</Steps>
