title: Game Rules
---

The "GameRule" content type in Strapi CMS is used to render a list of dropdowns on your website, containing information about different game rules provided by various vendors. You can create, modify, and delete entries to showcase game rules effectively.
<Tip>
  The updated content will be reflected in the burger menu right away. If it's not updating, you can perform a hard reload by pressing "Ctrl + Shift + R."
</Tip>

## Add New Vendor

<Steps>
  <Step title="Access the Content Manager">
    Log in to your Strapi CMS account and navigate to the Content Manager by clicking "Content Manager" in the dashboard sidebar. This is where you can manage your content types.
  </Step>
  <Step title="Select 'Game Rules'">
    In the Content Manager, find and click on "GameRule" in the list of content types.
  </Step>
  <Step title="Create a New Game Rule">
    To add a new game rule (vendor), click the "Add New Entry" button.
  </Step>
  <Step title="Fill in Game Rule Details">
    Fill in the following details for your new game rule:
    - **VendorName**: Enter the name or title of the vendor.

    <Note>Doing this will add a new vendor on the website with no games attached to it</Note>
  </Step>
  <Step title="Save Your Changes">
    After entering the vendor's details, click the "Save" button to create the new game rule.
  </Step>
</Steps>


## Create New Game List

<Steps>
  <Step title="Access the Content Manager">
    Log in to your Strapi CMS account and navigate to the Content Manager by clicking "Content Manager" in the dashboard sidebar.
  </Step>
  <Step title="Select 'GameRule'">
    In the Content Manager, find and click on "GameRule" in the list of content types.
  </Step>
  <Step title="Locate the Game Rule (vendor)">
    You'll see a list of existing game rules (vendors). Click on the game rule you want to add a game list to.
  </Step>
  <Step title="Create a New Game List">
    In the game rule's detail view, scroll down and click the "Add New Entry" button.

    <Note>This will create a new game list for the specified vendor with no games attached to it.</Note>
  </Step>
  <Step title="Save Your Changes">
    After filling in the game list details, click the "Save" button to create the new game list.
  </Step>
</Steps>


## Add Game

<Steps>
  <Step title="Access the Content Manager">
    Log in to your Strapi CMS account and navigate to the Content Manager by clicking "Content Manager" in the dashboard sidebar.
  </Step>
  <Step title="Select 'GameRule'">
    In the Content Manager, find and click on "GameRule" in the list of content types.
  </Step>
  <Step title="Locate the Game Rule (vendor)">
    You'll see a list of existing game rules (vendors). Locate the game rule you want to edit and click on it.
  </Step>
  <Step title="Add Game Properties">
    Once you've selected the game rule, you can make the following modifications:
    - **VendorName**: Update the vendor's name or title if needed.
    - **GameList**: Modify the gameList of the vendor as required.

    If you wish to add more games press the "Add new entry" button.
  </Step>
  <Step title="Save Your Changes">
    After making your desired edits, ensure to save your changes by clicking the "Save" button.
  </Step>
</Steps>

## How to Delete a Game Rule
<Warning>Deleting a Game Rule (vendor) will delete all games attached to it</Warning>
<Steps>
  <Step title="Access the Content Manager">
    Log in to your Strapi CMS account and navigate to the Content Manager by clicking "Content Manager" in the dashboard sidebar.
  </Step>
  <Step title="Select 'Game Rules'">
    In the Content Manager, find and click on "Game Rules" in the list of content types.
  </Step>
  <Step title="Locate the Game Rule to Delete">
    You'll see a list of existing game rules (vendors). Locate the game rule you want to delete and click on it.
  </Step>
  <Step title="Delete the Game Rule">
    In the game rule's detail view, look for the "Delete" or "Trash" button.
  </Step>
  <Step title="Confirm Deletion">
    When prompted, confirm the deletion. Please note that deleting a game rule is permanent and cannot be undone. Ensure that you no longer need the game rule before confirming the deletion.
  </Step>
</Steps>
