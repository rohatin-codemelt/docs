# Managing Burger Menu Items in Strapi CMS

The burger meniu items are displayed on the website in the left meniu on both mobile/desktop and is used to navigate throw the website pages.

- [3. Editing an Existing Burger Menu Item](#editing-an-existing-burger-menu-item)

## 1. Accessing Content Manager

Before you can manage Burger Menu Items, you need to access the Content Manager in Strapi. Here's how you can get there:

1. Log in to your Strapi CMS account.

2. In the dashboard, click on "Content Manager" in the sidebar.

## 2. Creating a New Burger Menu Item

To add a new item to your website's burger menu, follow these steps:

1. In the Content Manager, locate and click on "Burger Menu Items" in the list of content types.

2. You'll see a list of existing Burger Menu Items. To create a new one, click the "Add New Burger Menu Item" button.

3. Fill in the following details for your new item:
   - **Icon**: Upload an image for the menu item.
   - **Title**: Enter the text you want to display alongside the icon.
   - **Path**: Provide the path to a different page (e.g., '/casino', '/promotion').
   - **Position**: Specify the order in which this item should appear in the burger menu (e.g., 1, 2, 3).

4. After filling in the information, click the "Save" button to create the new Burger Menu Item.

## 3. Editing an Existing Burger Menu Item

If you need to make changes to an existing Burger Menu Item, follow these steps:

1. In the Content Manager, find and click on "Burger Menu Items" to see the list of existing items.

2. Locate the Burger Menu Item you want to edit and click on it.

3. Edit the properties as needed, such as the Icon, Title, Path, or Position.

4. Remember to save your changes by clicking the "Save" button.

## 4. Deleting a Burger Menu Item

To remove a Burger Menu Item from your website's burger menu, follow these steps:

1. In the Content Manager, find and click on "Burger Menu Items" to see the list of existing items.

2. Locate the Burger Menu Item you want to delete and click on it.

3. In the item's detail view, find the "Delete" or "Trash" button.

4. Confirm the deletion when prompted.

Please note that deleting a Burger Menu Item is permanent and cannot be undone.
