## 2. Creating a New Burger Menu Item

To add a new item to your website's burger menu, follow these steps:

1. In the Content Manager, locate and click on "Burger Menu Items" in the list of content types.

2. You'll see a list of existing Burger Menu Items. To create a new one, click the "Add New Burger Menu Item" button.

3. Fill in the following details for your new item:
   - **Icon**: Upload an image for the menu item.
   - **Title**: Enter the text you want to display alongside the icon.
   - **Path**: Provide the path to a different page (e.g., '/casino', '/promotion').
   - **Position**: Specify the order in which this item should appear in the burger menu (e.g., 1, 2, 3).

4. After filling in the information, click the "Save" button to create the new Burger Menu Item.

## 3. Editing an Existing Burger Menu Item

If you need to make changes to an existing Burger Menu Item, follow these steps:

1. In the Content Manager, find and click on "Burger Menu Items" in the list of content types.

2. You'll see a list of existing Burger Menu Items. Locate the item you want to edit and click on it.

3. Once you've selected the item, you can edit the properties as needed:

   - **Icon**: You can update the image representing the menu item. Upload a new image to replace the existing one, if necessary.

   - **Title**: Edit the text that is displayed alongside the icon. Modify it to reflect the updated menu item information.

   - **Path**: Change the path to a different page if required. Update the path to navigate users to the desired page.

   - **Position**: Modify the order in which this item should appear in the burger menu. Adjust the position value (e.g., from 1 to 2) to rearrange the items.

4. After making your edits, remember to save your changes by clicking the "Save" button.

5. The updated content will be reflected in the burger menu right away. If it's not updating, you can perform a hard reload by pressing "Ctrl + Shift + R."

By following these steps, you can easily edit existing Burger Menu Items to keep your website's navigation up to date and user-friendly.
